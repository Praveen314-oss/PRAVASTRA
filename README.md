## 🚀 What is PravAstra?

PravAstra is a feature-rich, single-page web application designed to be the **ultimate resource hub** for anyone pursuing a career in tech. From DSA practice and mock interviews to GATE prep, hackathon guides, and curated AI tools — everything lives in one beautifully crafted, neon-cyberpunk interface.

## 🛠️ Getting Started

### 🐧 Linux / macOS

```bash
# 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/PravAstra.git

# 2. Navigate into the folder
cd PravAstra

# 3. Open in your default browser
xdg-open index.html          # Linux
open index.html               # macOS
```

**Run a local server (recommended):**

```bash
# Using Python 3 (usually pre-installed)
python3 -m http.server 5500

# Using Python 2
python -m SimpleHTTPServer 5500

# Using Node.js (if installed)
npx serve .
```

Then open **http://localhost:5500** in your browser.

### 💡 VS Code (Easiest — Works on All OS)

1. Install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension
2. Open the `PravAstra` folder in VS Code
3. Right-click `index.html` → **"Open with Live Server"**
4. Auto-opens at `http://127.0.0.1:5500`

---

### 🌐 Deploy to GitHub Pages

```bash
# 1. Initialize git (if not already)
git init
git add .
git commit -m "🚀 Initial commit — PravAstra"

# 2. Push to GitHub
git remote add origin https://github.com/YOUR_USERNAME/PravAstra.git
git branch -M main
git push -u origin main
```

Then on GitHub:
1. Go to **Settings → Pages**
2. Set source to `main` branch, `/ (root)` folder
3. Click **Save** — your site goes live at:  
   `https://YOUR_USERNAME.github.io/PravAstra/`

## 🧰 Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | App structure and semantic markup |
